Numfit is an incredibly feature-rich numerical data processing library for JavaScript and NodeJS. It boasts a comprehensive collection of functions and classes for numerical interpolation, extrapolations, regression, and smoothing, supporting multiple dimensions and catering to a wide range of applications in graphics, finance, data analysis, machine learning, and many more! Numfit is meticulously tested, ensuring its efficiency and ease of use. So, why not give Numfit a try and experience its remarkable capabilities for yourself?

import { Linear } from 'Numfit';

const linear = new Linear( [ .35, .7 ], [ 2, 4 ] );

const start =  0;
const end = 1;
const stepSize = 0.025;

const points = [];

for ( let x = start; x <= end; x += stepSize ) {

	const y = linear.evaluate( x );
	points.push( [ y, x ] );

}
