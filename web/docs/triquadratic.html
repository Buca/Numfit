Triquadratic

Edit | Source

The Triquadratic class provides tools for performing three dimensional quadratic interpolation aka triquadratic interpolation and extrapolation, which are foundational techniques in data science for estimating values within or beyond a range of known data points. Interpolation employs biquadratic polynomial to construct new data points within the range of known values, essentially bridging the gaps between discrete points. In contrast, extrapolation extends the quadratic relationship beyond the known data, enabling predictions outside the given range. These techniques are widely employed in various applications, including signal processing, image analysis, and scientific simulations.

The Triquadratic() constructor creates a new instance of the Triquadratic class. It takes three parameters:

The constructor initializes the Triquadratic instance with the provided positions and values. If the coefficients parameter is not provided, it will be calculated internally.

The positions property of the Triquadratic instance represents the array of positions corresponding to the known data points. It is accessible using dot notation and returns an array of length 9 containing the numerical positions. The format of the positions array is: [x0, y0, z0, x1, y1, z1, x2, y2, z2].

.step( startX, startY, startZ, endX, endY, endZ, sizeX, sizeY, sizeZ: Number ) : Array

The step() method evaluates the triquadaratic equation at a specified number of equally spaced intervals between a given the starting values startX, startY and startZ and the ending values endX, endY and endZ. It returns an array of interpolated values corresponding to the evaluation points.

.apply( positions: Array ): Array

The apply() method evaluates the triquadratic equation for each value in a provided array of positions positions. It returns an array of corresponding values for each position. The array of positions should be formatted as [x0, y0, z0, x1, y1, z1, x2, y2, z2, ...], where (x0, y0, z0), (x1, y1, z1), (x2, y2, z2), etc. are the points at which to evaluate the triquadratic equation.
