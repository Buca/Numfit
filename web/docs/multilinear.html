Multilinear

Edit | Source

The Trilinear class provides tools for performing three dimensional linear interpolation aka trilinear interpolation and extrapolation, which are foundational techniques in data science for estimating values within or beyond a range of known data points. Interpolation employs bilinear polynomials to construct new data points within the range of known values, essentially bridging the gaps between discrete points. In contrast, extrapolation extends the linear relationship beyond the known data, enabling predictions outside the given range. These techniques are widely employed in various applications, including signal processing, image analysis, and scientific simulations.

Static Properties

Multilinear.degree: Number

The maximum degree of the free variables of the multilinear equation. Returns 1.

Static Methods

Multilinear.evaluate( positions, values, position: Array ): Number

The Multilinear.evaluate() is a static method that evaluates the multilinear equation for the given input position. It takes three arguments:

positions: A numerical array specifies the positions of the values to be interpolated. The format of the array is: [

    p0,0, p1,0, ..., pN-1,0,
    p0,1, p1,1, ..., pN-1,1

], where N is the number of free variables in the multilinear equation.
values: For example the format of the values array for a multilinear equation with 4 free variables is: [

    v0000, v1000,
    v0100, v1100,

    v0010, v1010,
    v0110, v1110,


    v0001, v1001,
    v0101, v1101,

    v0011, v1011,
    v0111, v1111

]. Genrally the length of the values array is (D+1)N Where D is the the degree of the polynomial equation and N is the number of free variables. In the case of multilinear equation it is 2N.
position: The size of this array defines the number of free variables of the multilinear equation. The format of the values array is:
[ p0, p1, ..., pN-1 ].

The method returns the calculated value corresponding to the input value x based on the multilinear equation defined by the provided positions and corresponding values.
