Cubic

Edit | Source

The Cubic class provides tools for performing quadratic interpolation and extrapolation. Interpolation employs quadratic equations to construct new data points within the range of known values, essentially bridging the gaps between discrete points. In contrast, extrapolation extends the cubic relationship beyond the known data, enabling predictions outside the given range.

Constructor

new Cubic( positions, values, ?coefficients: Array ): Cubic

The Cubic() constructor creates a new instance of the Cubic class. It takes three parameters:

positions: A numerical array of length 4 that specifies the positions of the values to be interpolated.
values: A numerical array of length 4 that contains the values to be interpolated.
coefficients: (Optional) An optional numerical array of coefficients used internally for evaluating interpolated values.

The constructor initializes the Quadratic instance with the provided positions and values. If the coefficients parameter is not provided, it will be calculated internally.

Static Properties

Cubic.variables: Number

The number of free variables. Since the Cubic equation is a monomial - it is 1.

Cubic.degree: Number

The maximum degree of the polynomial. In the case of Cubic it is 3.

Static Methods

Cubic.coefficients( x0, x1, x2, x3, v0, v1, v2, v3: Number, output: Array ): Array

The Cubic.coefficients() static method determines the coefficients of the cubic equation that passes through the four points (x0, v0), (x1, v1), (x2, v2) and (x3, v3). It takes nine arguments:

x0: The x-coordinate of the first point.
x1: The x-coordinate of the second point.
x2: The x-coordinate of the third point.
x3: The x-coordinate of the fourth point.
v0: The y-coordinate of the first point.
v1: The y-coordinate of the second point.
v2: The y-coordinate of the third point.
v3: The y-coordinate of the fourth point.
output: (Optional) An optional array to store the calculated coefficients. If not provided, a new array will be created.

The method returns an array containing the two coefficients of the quadratic equation: [c0, c1, c2, c3 ], where the equation is expressed as v = c0 + c1x + c2x2 + c3x3.

Cubic.evaluate( x0, x1, x2, x3, v0, v1, v2, v3, x: Number ): Number

The Cubic.evaluate() static method evaluates the quadratic equation that passes through the four points (x0, v0), (x1, v1), (x2, v2) and (x3, v3) for a given input value x. It takes nine arguments:

s
x0: The x-coordinate of the first point.
x1: The x-coordinate of the second point.
x2: The x-coordinate of the third point.
x3: The x-coordinate of the fourth point.
v0: The y-coordinate of the first point.
v1: The y-coordinate of the second point.
v2: The y-coordinate of the third point.
v3: The y-coordinate of the fourth point.
x: The position to evaluate the cubic equation.

The method returns the calculated y-value corresponding to the input value x based on the cubic equation defined by the provided points.

Instance Properties

.positions: Array

The positions property of the Cubic instance represents the array of positions corresponding to the known data points. It is accessible using dot notation and returns an array of length 4 containing the numerical positions.

The positions property is mutable, meaning that changes made to the returned array will also be reflected in the internal positions of the Cubic instance. This allows for dynamic modification of the interpolation range.

.values: Array

The values property of the Cubic instance represents the array of values corresponding to the known data points. It is accessible using dot notation and returns an array of length 4 containing the numerical values.

The values property is mutable, meaning that changes made to the returned array will also be reflected in the internal values of the Cubic instance. This allows for dynamic modification of the interpolation range.

.coefficients: Array

The coefficients property of the Cubic instance represents the array of coefficients used internally for evaluating interpolated values. It is accessible using dot notation and returns an array of length 4 containing the numerical coefficients.

To recompute the internal coefficients, you need to call the update() method on the Cubic instance.

Instance Methods

.update(): Cubic

The update() method recalculates the coefficients. This should be used after performing a translation, scaling or when adjusting either the positions or values of the instance.

.translate( t: Number ): Cubic

The translate() method translates the positions of the Cubic instance by a specified value t. This means that it adds t to each position. It returns the instance itself.

t: The value by which to translate the positions.

.scale( s, ?anchor: Number ): Cubic

The scale() method scales the positions of the Cubic instance around an optional anchor point anchor by a specified factor s. If no anchor point is provided, the scaling is done with respect to the origin 0. It returns the instance itself, allowing for chaining operations.

s: The scaling factor to apply to the positions.
anchor: An (optional) anchor point around which to scale the positions. If omitted, the origin 0 is used.

.evaluate( x: Number ): Number

The evaluate() method evaluates the cubic equation for a given input value x. It returns the corresponding value based on the cubic relationship defined by the instance's positions and values.

x: The x-value for which to evaluate the cubic equation.

.step( start, end, size: Number ) : Array

The step() method evaluates the cubic equation at a specified number of equally spaced intervals between a given start value start and end value end. The number of intervals is determined by the step size size, which specifies the distance between each evaluation point. It returns an array of interpolated values corresponding to the evaluation points.

start: The starting value of the interpolation range.
end: The ending value of the interpolation range.
size: The step size, which determines the distance between evaluated points and the number of evaluation points between the start and end values.

.segment( start, end, segments: Number ): Array

This method calculates interpolated values between specified start and end points, dividing the interval into a given number of segments. It takes three parameters:

start: The starting value of the interpolation range.
end: The ending value of the interpolation range.
segments: The number of segments into which the range should be divided.

The method returns an array of interpolated values, with each element corresponding to the value at a specific segment point. It utilizes cubic interpolation to estimate the values within each segment.

.apply( positions: Array ): Array

The apply() method evaluates the cubic equation for each value in a provided array of positions positions. It returns an array of corresponding values for each position. The array of positions should be formatted as [x0, x1, x2, ...], where x0, x1, x2, etc. are the positions at which to evaluate the cubic equation.

positions: An array of positions at which to evaluate the cubic equation. The array should be formatted as [x0, x1, x2, ...], where x0, x1, x2, etc. are the positions.
