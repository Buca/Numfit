The Tricubic class provides tools for performing three dimensional cubic interpolation and extrapolation.

Instance Methods

.update(): Tricubic

The update() method recalculates the coefficients. This should be used after performing a translation, scaling or when adjusting either the positions or values of the instance.

.translate( offset ): Tricubic

The translate() method translates the positions of the instance by the values of the offset array. It returns the instance itself.

offset: Array An array of two values by which to offset the positions. The layout of the offset array is: [ offsetx, offsety, offsetz ], where offsetx, offsety and offsetz are the values by which to translate in the x, y and z directions respectively.

.scale( scalars, ?orgin ): Tricubic

The scale() method scales the positions of the instance around an optional orgin point. If no orgin point is provided, the scaling is done with respect to the origin [ 0, 0, 0 ]. It returns the instance itself, allowing for chaining operations.

scalars: Array The scaling values by which to scale the positions. The layout of the array is: [ scalarx, scalarx, scalarx ].
orgin: Array (Optional) The point around which to scale positions. The layout of the array is [ orginx, orginx, orginx ]. Defaults to [ 0, 0, 0 ].

.evaluate( position ): Number

The evaluate() method evaluates the tricubic equation for a given position. It returns the corresponding evaluated value based on the tricubic relationship defined by the instance's positions and values.

position: Array The coordinates for which to evaluate the tricubic equation. The layout of the array is: [ x, y, z ].

.step( start, end, size ): Array

The step() method evaluates the tricubic equation at a specified number of equally spaced intervals between given starting values and ending values. The number of intervals is determined by the step-sizes sizeX and sizeY, which specifies the distance between each evaluation point. It returns an array of interpolated values corresponding to the evaluation points.

start: Array The starting coordinates of the interpolation range. The layout of the array is: [ startx, starty, startz ].
end: Array The ending coordinates of the interpolation range. The layout of the array is [ endx, endx, endx ].
size: Array The step-sizes to take between points. The layout of the array is: [ sizex, sizey, sizez ].

.segment( start, end, segments ): Array

This method calculates interpolated values between specified start and end points, dividing the interval into a given number of segments. It takes six parameters:

start: Array The starting coordinates of the interpolation range. The layout of the array is: [ startx, starty, startz ].
end: Array The ending coordinates of the interpolation range. The layout of the array is [ endx, endx, endx ].
segments: Array The amount of segments take points between the starting and ending points. The layout of the array is: [ segx, segy, segz ].

The method returns an array of interpolated values, with each element corresponding to the value at a specific segment point. It utilizes tricubic interpolation to estimate the values within each segment.

.apply( positions ): Array

The apply() method evaluates the tricubic equation for each value in a provided array of positions positions. It returns an array of corresponding values for each position.

positions: Array An array of positions at which to evaluate the tricubic equation. The layout of the positions array is: [

     x0,  y0, z0,
     x1,  y1, z1,
     x2,  y2, z2,
      ⋮    ⋮
    xn-1, yn-1, zn-1

], where n is the number of points and [ x0, y0, z0 ], [ x1, y1, z1], [ x2, y2, z2 ], …, [ xn-1, yn-1, zn-1 ] are the points at which to evaluate the tricubic equation(s).
