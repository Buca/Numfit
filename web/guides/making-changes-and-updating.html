Making Changes & Updating

Edit

This guide will show how to make changes to the positions and values on an evaluator and how to update the evaluator so that the changes are reflected on evaluation.

Using translate, scale and update functions

Using the .translate(), .map() and .update() functions on an instance returns that instance, so they can be chain. Generally you want to make all your calls to .translate() and .map() before finally calling .update().

Here's example code where we translate, scale and then update an instance of the Linear class: 


const positions = [];
const values = [];		
const linear = new Linear( positions, values );

linear.translate( -1 ).scale( 10 ).update();
