Static Properties

Linear.variables: Number

The number of free variables. Returns 1.

Linear.degree: Number

The maximum degree of the polynomial. Returns 1.

Static Methods

Linear.coefficients( positions, values: Array, ?dimension: Number, ?output: Array, ?outputOffset: Number ): Array

The Linear.coefficients() static method calculates the coefficients of the linear equations that pass through the two points (x0, v0i) and (x1, v1i) for each 0 ≤ i ≤ d-1, where d is the dimension of the value-space. It takes five arguments:

positions: A numerical array of length 2 that specifies the positions of the values to be interpolated. The layout of the array is [ x0, x1].
values: A numerical array of length d × 2 containing the values to be interpolated. The layout of the values array is: [

    v00, v10, …, v(d-1)0,
    v01, v11, …, v(d-1)1

], where d is the dimension of the value-space.
dimension: (Optional) A numerical value that describes the dimension of the value-space. Defaults to 1.
output: (Optional) An array to store the calculated coefficients. If not provided, a new array will be created. The layout of the output array is: [

     c00,     c10,
     c01,     c11,
      ⋮        ⋮
    c0(d-1), c1(d-1)

], where d is the dimension of the values space and the first index corresponds to the exponent of the free variables.
outputOffset: (Optional) A positive integer which describes how much to offset the index layout in the output.

The method returns an array containing the the coefficients of the d linear equations: [

     c00,     c10,
     c01,     c11,
      ⋮        ⋮
    c0(d-1), c1(d-1)

], where the ith linear equation is expressed as v = c1ix + c0i for 0 ≤ i ≤ d-1 where d is the dimension of the value-space.

Linear.evaluate( positions, values, position: Array, ?dimension: Number, output: Array, outputOffset: Number ): Array

The Linear.evaluate() static method evaluates the linear equation(s) which pass through the points (x0, vi0) and (x1, vi1) for each 0 ≤ i ≤ d-1, where d is the dimension of the value-space. It takes five arguments:

positions: A numerical array of length 2 that specifies the positions of the values to be interpolated. The layout of the array is [ x0, x1 ].
values: A numerical array of length d × 2 containing the values to be interpolated. The layout of the is: [

    v00, v10, …, v(d-1)0,
    v01, v11, …, v(d-1)1

], where d is the dimension of the value-space. The value at position xj is vij, for some integers i and j such that 0 ≤ i ≤ d-1 and 0 ≤ j ≤ 1.
position: A numerical array of length 1 containing the coordinate value to be interpolated. The layout of the position array is: [ x ].
dimension: (Optional) A numerical value that describes the dimension of the value-space. Defaults to 1.
output: (Optional) A numerical array of size d to store the evaluated values, where d is the dimension of the value-space. If not provided, a new array will be created. The layout of the output array is: [ k0, k0, … kd-1 ], where d is the dimension of the value-space and the first index corresponds to the exponent of the free variables.
outputOffset: (Optional) A positive integer which describes how much to offset the index layout in the output.

The method returns the calculated value corresponding to the input value x based on the linear equation defined by the provided points.

Instance Properties

.positions: Array

The positions property of the Linear instance represents the array of positions corresponding to the known data points. It is accessible using dot notation and returns an array of length 2 containing the numerical positions. The layout of the values array is: [ x0, x1 ].

.values: Array

The values property of the Linear instance represents the array of values corresponding to the known data points. Returns an array of length d × 2 containing the numerical values, where d is the dimension of the value-space.

The layout of the values array is: [

    v00, v10, ..., v(d-1)0,
    v01, v11, ..., v(d-1)1

], where d is the dimension of the value-space.

.coefficients: Array

The coefficients property of the Linear instance represents the array of coefficients used internally for evaluating interpolated values. It is accessible using dot notation and returns an array of length d × 2 containing the numerical coefficients, where d is the dimension of the value-space. It should be treated as read-only.

The layout of the coefficients array is: [

     c00,     c10,
     c01,     c11,
      ⋮        ⋮
    c0(d-1), c1(d-1)

], where d is the dimension of the value-space.

.dimension: Number

The dimension property of the Linear instance represents the dimension of the value-space. It should treated as read-only.

Instance Methods

.update(): Linear

The update() method recalculates the coefficients. This should be used after performing a translation, scaling or when adjusting either the positions or values of the instance.

.translate( t: Array ): Linear

The translate() method translates the positions of the Linear instance by a specified value t. This means that it adds t to each position. It returns the instance itself.

t: The value by which to translate the positions.

.scale( s, ?anchor: Array ): Linear

The scale() method scales the positions of the Linear instance around an optional anchor point anchor by a specified factor s. If no anchor point is provided, the scaling is done with respect to the origin 0. It returns the instance itself, allowing for chaining operations.

s: The scaling factor to apply to the positions.
anchor: An (optional) anchor point around which to scale the positions. If omitted, the origin 0 is used.

.evaluate( position: Array | Number, inputOffset: Number, output: Array, outputOffset: Number ): Array

The evaluate() method evaluates the linear equation(s) for a given position [ x ]. It returns an array with the corresponding value(s).

position: The position to be evaluated. The layout of the position array is: [ x ].
inputOffset: (Optional) The index offset for the position array. Defaults to 0.
output: (Optional) The output array for the evaluated values. An array will be created if none is provided. The layout of the output array is: [ k0, k1, ..., kd-1 ], where d is the dimension of the value-space.
outputOffset: (Optional) The index offset for the output array. Defaults to 0.

.step( start, end, size: Array | Number, ?handler: Function ) : Array | Linear

The step() method evaluates the linear polynomial at a specified number of equally spaced intervals between a given start value start and end value end. The number of intervals is determined by the step size size, which specifies the distance between each evaluation point. It returns an array of interpolated values corresponding to the evaluation points.

start: The array containing the starting value of the interpolation range. The layout of the array is: [ s ], where s is the starting value.
end: The array containing the ending value of the interpolation range. The layout of the array is: [ e ], where e is the ending value.
size: The array containing the step size in the interpolation range. The layout of the array is: [ stepSize ], where stepSize is the step size.
handler: (Optional) A function to handle each point. Where the handler function is given x and v as arguments. If the handler function is provided then map() returns this instance.

.segment( start, end, segments: Array | Number, ?handler: Function ): Array | Linear

This method calculates interpolated values between specified start and end points, dividing the interval into a given number of segments. It takes three parameters:

start: The array containing the starting value of the interpolation range. The layout of the array is: [ s ], where s is the starting value.
end: The array containing the ending value of the interpolation range. The layout of the array is: [ e ], where e is the ending value.
segments: The array containing the number of segments in the interpolation range. The layout of the array is: [ segs ], where segs is the number of segments.
handler: (Optional) A function to handle each point. Where the handler function is given x and v as arguments. If the handler function is provided then segment() returns this instance.

The method returns an array of interpolated values, with each element corresponding to the value at a specific segment point.

.map( positions: Array, ?handler: Function ): Array | Linear

The map() method evaluates the linear equation(s) for each value in a provided array of positions positions. It returns an array of corresponding values for each position. The array of positions should be formatted as [ x0, x1, x2, ... ], where x0, x1, x2, etc. are the positions at which to evaluate the linear equation(s).

positions: An array of positions at which to evaluate the linear equation(s). The positions layout is: [x0,x1, x2, ...], where x0, x1, x2, etc. are the positions.
handler: (Optional) A function to handle each point. Where the handler function is given x and v as arguments. If the handler function is provided then map() returns this instance.
