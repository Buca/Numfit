Basic Evaluation

Edit

This guide will describe how to use both static and instance evaluation.

Static evaluation

Static evaluation requires very little setup other than defining the positions, values and optionally the dimension of the value-space. Here's example code showing how to use static evaluation to linearly interpolate halfway between two RGBA values using the Linear class: 


const positions = [ 0, 1 ];
const values = [ 0,0,0,0, 255,255,255,1 ];
const dimension = 4;

Linear.evaluate( positions, values, dimension, 0.5 );

Since static evaluation calculates the values directly, it is best suited for when you only need one or a few values to be evaluated. Instance evaluation is better suited for when you need more than one evaluation, because it uses coefficients to evaluate.

Static evaluation

Instance evaluation requires a bit more setup: defining the positions, values, optionally the dimension of the value-space and additionally constructing the instance. Here's example code showing how to use instance evaluation to linearly interpolate between two RGBA values using the Linear instance: 


const positions = [ 0, 1 ];
const values = [ 0,0,0,0, 255,255,255,1 ];
const dimension = 4;

const linear = new Linear( position, values, dimension );

linear.evaluate( 0.5 );
