Using Samplers

Edit

This guide will describe how to use different types of samplers. There are currently three different samplers: .step, .segment and .map

Using the Step sampling function

At it's simplest the function .step( start, end, step ) takes three arguments: the starting value of the evaluation range start, the ending value end and the step-size size.

Here's example code showing how to use the .step() function to interpolate between two RGBA values within the range defined by start and end according to a step-size defined by size: 


const positions = [ 0, 1 ];
const values = [ 0,0,0,0, 255,255,255,1 ];
const dimension = 4;

const linear = new Linear( positions, values, dimension );

linear.step( -1, 2, 0.25 );

If we instead use an evalauator class with a number of free-variables higher than one, like Bilinear, Trilinear, then we use arrays of values for the sampler's arguments. Here's example code of the .step() function using the Trilinear class instead: 


const positions = [ 
	0, 1,
	0, 1,
	0, 1
];
const values = [
	1, 2,
	3, 4,

	5, 6,
	7, 8
];
const trilinear = new Trilinear( positions, values );

const start = [ 0, 1, 2 ];
const end = [ 2, 3, 4 ];
const size = [ 0.2, 0.2, 0.2 ];

const vs = trilinear.step( start, end, size );

Using the Segment sampling function

The segment function .segment( start, end, step ), like the .step() function, takes three arguments: the starting value of the evaluation range start, the ending value end and the number of segments in the range segments.

Here's example code showing how to use the .map() function to interpolate between two RGBA values within the range defined by start and end according to a along number of equally spaced segments defined by segments argument: 


const positions = [ 0, 1 ];
const values = [ 0,0,0,0, 255,255,255,1 ];
const dimension = 4;

const linear = new Linear( positions, values, dimension );

const vs = linear.segment( -1, 2, 12 );

Using the Map sampling function

The map function .map( positions ) takes one argument: the positions array.

Here's example code showing how to use the .map() function to evaluate positions: 


const positions = [ 0, 1 ];
const values = [ 0,0,0,0, 255,255,255,1 ];
const dimension = 4;

const linear = new Linear( positions, values, dimension );

const ps = [ -1, 3, 1, 5 ];
const vs = linear.map( ps );
